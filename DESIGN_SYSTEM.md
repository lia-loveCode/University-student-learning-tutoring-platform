# Design System (UED 设计规范)

本项目采用「浅色内容区 + 深色顶部/首屏」的教育平台风格，参考学习通 PC、学堂在线、学银在线、MOOC 的信息架构与视觉气质。

## 1. 设计规范参数（Design Tokens）

所有基础规范以 CSS 变量形式维护在：

- `src/styles/tokens.css`

包含：

- **颜色**：主色（教育蓝）、中性色（文字/背景/边框）、状态色、首屏装饰色（蓝紫）
- **字号与行高**：12/14/16/18/20/24/32/40
- **圆角**：6/8/12/16/pill
- **间距**：8pt 系统（4/8/12/16/20/24/32/40/48/64）
- **阴影与 focus-ring**
- **版心**：`--container-max`、`--container-padding-x`

## 2. 使用规则（重要）

- **不要在组件里写“散装值”**：优先使用 token（例如 `var(--color-primary)`、`var(--space-16)`）。
- **先统一基础，再做页面**：按钮、输入框、卡片、标签等应复用同一套 token，保证全站一致。
- **深色只用于 Header/Hero**：内容区建议保持浅色，提高可读性与信息密度。

## 3. 推荐的页面默认值

- **正文默认**：14px
- **页面背景**：`var(--color-bg)`
- **卡片背景**：`var(--color-surface)`，圆角 12px，阴影 `--shadow-card`
- **按钮圆角**：8px（胶囊按钮用 pill）

## 4. 后续迁移建议

当前项目已有部分历史变量（例如 `--bg`、`--text`、`--accent`）。`tokens.css` 已提供 alias，方便逐步迁移。
